# huntd-testing
Manual testing of Huntd web and mobile application.

## Scope
Testing of main user flows:
- user registration
- login
- job search
- navigation between pages

## Test documentation
Test cases were created in TestRail and exported as XML.

Repository structure:
- huntd_web.xml – test cases for web application
- huntd_mobile.xml – test cases for mobile application

## Testing techniques
- test case design
- positive and negative testing
- UI testing
- basic regression testing
